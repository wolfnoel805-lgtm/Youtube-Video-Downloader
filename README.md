# YouTube Downloader by elyxnoel

Ein moderner, schlanker und extrem schneller YouTube Video Downloader mit einer schicken grafischen Benutzeroberfläche (GUI). 

## Features

- **Modernes UI:** Dunkles, minimalistisches Design basierend auf CustomTkinter.
- **Echte Progressbar:** Live-Anzeige des Download-Fortschritts in Prozent.
- **Echtzeit-Statistiken:** Zeigt die aktuelle Download-Geschwindigkeit und die verbleibende Zeit (ETA) an.
- **Einfügen-Button:** Kopierte YouTube-Links können mit einem Klick direkt aus der Zwischenablage eingefügt werden.
- **Freie Ordnerwahl:** Du entscheidest bei jedem Download selbst, wo das Video gespeichert wird.
- **Standalone-Modus:** Die Eingabeaufforderung (CMD) bleibt beim Starten komplett unsichtbar.
