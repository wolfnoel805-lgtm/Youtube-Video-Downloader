# YouTube Downloader by elyxnoel

Ein moderner, schlanker und extrem schneller YouTube Video Downloader mit einer schicken grafischen Benutzeroberfläche (GUI). 

## Features

- **Modernes UI:** Dunkles, minimalistisches Design basierend auf CustomTkinter.
- **Echte Progressbar:** Live-Anzeige des Download-Fortschritts in Prozent.
- **Echtzeit-Statistiken:** Zeigt die aktuelle Download-Geschwindigkeit und die verbleibende Zeit (ETA) an.
- **Einfügen-Button:** Kopierte YouTube-Links können mit einem Klick direkt aus der Zwischenablage eingefügt werden.
- **Freie Ordnerwahl:** Du entscheidest bei jedem Download selbst, wo das Video gespeichert wird.
- **Standalone-Modus:** Die Eingabeaufforderung (CMD) bleibt beim Starten komplett unsichtbar.

## Voraussetzungen

Stelle sicher, dass du Python installiert hast. Bevor du das Programm startest, musst du die benötigten Erweiterungen installieren. Öffne dazu deine Befehlszeile (CMD) und gib Folgendes ein:

```bash
pip install yt-dlp customtkinter
Installation & Benutzung
Klicke oben rechts auf den grünen Code-Button und wähle Download ZIP.

WICHTIG: Klicke mit rechts auf die heruntergeladene ZIP-Datei und wähle Alle extrahieren..., um den Ordner vollständig zu entpacken. (Das Programm läuft nicht direkt aus der ZIP-Datei!).

Öffne den entpackten Ordner und mache einen Doppelklick auf die Datei Starten.bat.

Das Programm öffnet sich direkt (ohne dass im Hintergrund eine schwarze CMD-Box zu sehen ist).

Kopiere einen YouTube- oder Shorts-Link, klicke im Programm auf Einfügen und drücke auf DOWNLOAD.

Wähle den gewünschten Speicherort aus – der Download startet sofort!
